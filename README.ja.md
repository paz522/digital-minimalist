# Digital Minimalist

**スマホとの健康的な関係を築くための無料 Web アプリ**

![Digital Minimalist](https://via.placeholder.com/800x450/667eea/ffffff?text=Digital+Minimalist)

## 🌟 主な機能

- **📊 ダッシュボード** - 使用時間の追跡と目標管理
- **🎯 フォーカスモード** - ポモドーロテクニック対応タイマー
- **📈 アナリティクス** - 詳細な使用時間分析
- **🏆 アチーブメント** - 8 つのバッジでモチベーション維持
- **📔 振り返りジャーナル** - 気分記録と振り返り
- **📖 記事** - デジタルデトックスのガイド
- **⚙️ 設定** - カスタマイズとデータエクスポート

## 🚀 クイックスタート

```bash
npm install
npm run dev
```

## 📦 ビルド

```bash
npm run build
```
